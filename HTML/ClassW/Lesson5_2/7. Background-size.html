Фоновый рисунок страницы

Свойство background-size позволяет установить размер фонового изображения. Для установки размера можно использовать либо пиксели, либо проценты, либо одно из предустановленных значений:

  • contain: масштабирует изображение по наибольшей стороне
  • cover: масштабирует изображение по наименьшей стороне
  • auto: значение по умолчанию, изображение отображается в полный размер
